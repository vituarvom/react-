## capitalize

The `capitalize` function is a utility function that converts the first character of a given string to uppercase, leaving the rest of the string unchanged. This function is useful for formatting text in titles, names, and standardizing user input formats.

## Installation

Make sure to have the `string-smart-utils` library installed:

```bash
npm install string-smart-utils
